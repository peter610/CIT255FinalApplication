Title: Guns R Us
Description: A digital catalog of firearms with some basic information.
Application Type: Console
Persistence Type: SQL
Persistence Location: Local Database
Author: Scott Peterson
Date Created: November 24, 2017
Last Modified: December 10, 2017
Instructions for Use:
	1. Press any key to move passed the splash screen.
	2. Make a selection from the Main Menu using the numbers corresponding to the option.
	3. Follow the on-screen prompts for each menu choice.
	4. Press "e" to exit the application from the main menu.
